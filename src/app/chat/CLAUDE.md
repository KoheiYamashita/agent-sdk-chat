# src/app/chat/

チャット機能のページを格納しています。

## ファイル構成

| ファイル/ディレクトリ | 役割 |
|---------------------|------|
| `page.tsx` | チャット一覧ページ（新規チャット作成へリダイレクト） |
| `layout.tsx` | チャットレイアウト（サイドバー、ターミナルパネル統合） |
| `[sessionId]/` | 動的ルート（個別チャットセッション） |

## ルーティング

- `/chat` - チャット一覧（新規セッション作成）
- `/chat/[sessionId]` - 個別チャットセッション表示
