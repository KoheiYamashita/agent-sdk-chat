# src/app/settings/

設定ページを格納しています。

## ファイル構成

| ファイル/ディレクトリ | 役割 |
|---------------------|------|
| `page.tsx` | メイン設定ページ（APIキー、権限モード、デフォルトツール、外観設定） |
| `layout.tsx` | 設定ページレイアウト（サイドバー統合） |
| `models/` | カスタムモデル管理ページ |
| `skills/` | スキル管理ページ |
| `danger/` | 危険な設定ページ（使用量表示） |

## ルーティング

- `/settings` - 一般設定（APIキー設定含む）
- `/settings/models` - カスタムモデル管理
- `/settings/skills` - スキル管理
- `/settings/danger` - 危険な設定（個人利用向け）

## 危険な設定（Danger Zone）

`/settings/danger`では以下の設定を管理します：

### 使用量表示（`showUsage`）
- **デフォルト**: OFF
- **機能**: サイドバーに使用量表示へのリンクを追加
- **動作**:
  - OFF時: サイドバーに使用量リンク非表示、`/usage`アクセス時は`/settings/danger`にリダイレクト
  - ON時: サイドバーに使用量リンク表示、使用量ページにアクセス可能
- **有効化時**: 確認ダイアログを表示（ローカルClaude Code認証情報使用の注意事項）
